Order issue

Web-styled application usually have two entrance: Search and Browse.
When you input term and search, the result order will be descent by matched "score", normally it is fine.

But how about products have the same score? Lucene and a lot of search libraries sort them by index order. See stackoverflow question. That will be a problem, especially in Browse mode.
